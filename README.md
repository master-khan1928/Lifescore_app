# LifeScore - Self-Discipline App

Мобильное приложение для трекинга прогресса в жизни через систему баллов и уровней.

## Технологии
- React Native (Expo)
- TypeScript
- Zustand (State Management)
- React Navigation

## Запуск проекта
git clone https://github.com/master-khan1928/Lifescore_app.git
cd Lifescore_app
npm install
npm start
